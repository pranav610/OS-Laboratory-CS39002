# Shellkil 
## Dependency
- Readline library C
```bash
sudo apt-get install libreadline-dev
```
## Compile
```bash
    git clone github.com/pranav610/OS-Laboratory-CS39002
    cd Assignment2/shellfiles
    make
```
## Run
```bash
    ./shell
```
## Features
- delep command
```bash
    delep /path/to/file
```
- sb command: suggest flag is optional
```bash
    sb <pid> -suggest 
```
